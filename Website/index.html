จุดประสงค์

    กลุ่มของพวกเรามีขึ้นเพื่อจุดประสงค์ในการสร้างความบันเทิงและฝึกทักษะการแก้ปัญหา ทางผู้จัดทำได้ทำการพัฒนาโปรแกรมด้วยภาษา C โดยนำเสนอในรูปแบบเกมแนวการแก้ไขปัญหา (Puzzle) โดยผู้เล่นจะพบกับความท้าทายในการเล่นถึง 3 ระดับ ซึ่งจะมีความยากในการเล่นแตกต่างกันไป หลังจากพัฒนาชิ้นงานจนแล้วเสร็จทางเราได้ตั้งชื่อให้กับชิ้นงานชิ้นนี้ว่า "Neighbours"


    วิธีการเล่น

    ขึ้นชื่อว่าเป็นเกมแนว Puzzle ดังนั้นในเกมนี้ย่อมต้องมีปัญหาให้ผู้เล่นขบคิด ซึ่งปัญหาในเกมนี้คือ ชาวบ้านแต่ละคนจะมีความสัมพันธ์ที่แตกต่าง 2 รูปแบบ คืออาศัยร่วมกันได้ และอาศัยร่วมกันไม่ได้


    1.1 ชาวบ้านที่อาศัยร่วมกันไม่ได้


    1.2 ชาวบ้านอาศัยร่วมกันได้

    ซึ่งผู้เล่นจะได้มอบหมายให้จัดการวางลำดับของคนในหมู่บ้านให้ถูกต้อง ด้วยการนำชาวบ้านไปไว้ในตำแหน่งที่ต้องการ ถ้าหากชาวบ้านที่อาศัยอยู่ใกล้เคียงกันไม่สามารถอาศัยร่วมกันได้ ชาวบ้านคนนั้นจะแสดงท่าทางไม่พอใจ ดังนั้นผู้เล่นจะต้องเปลี่ยนตำแหน่งของชาวบ้าน จนกว่าชาวบ้านทุกคนจะสามารถอาศัยร่วมกันได้


    เมื่อผู้เล่นสามารถเอาชนะเกมได้แล้ว จะมีการแสดงจำนวนการเคลื่อนย้ายที่ผู้เล่นได้ทำการเคลื่อนย้ายชาวบ้านทั้งหมด และแสดงจำนวนที่น้อยที่สุดที่ผู้เล่นเคยทำได้เอาไว้




    ตัวอย่าง Code และ Algorithm

    ในผลงานชิ้นนี้พวกเราได้ใช้ภาษา C ในการสร้างเป็นหลัก โดยเลือกใช้ไลบรารี่ SDL ซึ่งมีจุดเด่นคือ มีความ low-level มากกว่า SFML หรือก็คือมีลูกเล่นที่หลายหลายมากกว่านั่นเอง นอกจากนี้ยังสามารถ cross-platform ทั้ง Windows, Linux, Mac, iOS

    2.1 ตัวอย่างโค้ดที่ใช้ในการสร้างหน้าต่างเกมในไฟล์ main.cpp(คลิกที่ภาพเพื่อดูโค้ดทั้งหมด)

    2.2 ตัวอย่างอัลกอริทึมที่ใช้ในการประมวณผลของตัวเกมในไฟล์ game.cpp(คลิกที่ภาพเพื่อดูโค้ดทั้งหมด)

    2.3 ตัวอย่างอัลกอริทึมที่ใช้ในการประมวณผลของตัวเกมในไฟล์ game.cpp(คลิกที่ภาพเพื่อดูโค้ดทั้งหมด)
